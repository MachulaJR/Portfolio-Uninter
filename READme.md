# 🎓 Portfólio Pessoal – Atividade Prática UNINTER

Portfólio online desenvolvido como atividade prática da disciplina **Fundamentos da Programação Web** do curso de **Análise e Desenvolvimento de Sistemas (UNINTER)**.

## 📋 Sobre o Projeto

O objetivo da atividade foi desenvolver um portfólio pessoal aplicando os conceitos de **HTML5**, **CSS3** e **JavaScript puro**, sem o uso de frameworks ou bibliotecas externas, com publicação obrigatória via **GitHub Pages**.

## 📄 Seções

- **Sobre mim** – Apresentação pessoal, informações de contato e hobbies
- **Formação** – Educação formal, certificações e linha do tempo acadêmica
- **Portfólio** – Projetos desenvolvidos com links para os repositórios
- **Contato** – Formulário com validação em JavaScript

## ⚙️ Funcionalidades JavaScript

- ✅ Validação completa do formulário de contato (nome, e-mail e mensagem)
- ✅ Simulação de envio com mensagem de confirmação
- ✅ Alternância de tema claro/escuro com persistência via `localStorage`
- ✅ Menu responsivo (hamburguer) para dispositivos móveis
- ✅ Link ativo no menu conforme a seção visível na tela

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (sem frameworks)
- JavaScript (sem frameworks)
